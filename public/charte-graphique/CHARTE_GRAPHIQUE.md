# Charte Graphique nSignal 2025

## Introduction

Cette charte graphique définit l'identité visuelle officielle de **nSignal**, la plateforme de pilotage social pour réseaux multi-sites.

**Slogan:** *"n locations. One signal."*

**Mission visuelle:** Représenter la connexion, la diffusion et la coordination à travers un réseau distribué.

---

## 1. Logo nSignal

### 1.1 Concept

Le logo nSignal représente:
- **Le "n"** central: le cœur du réseau (le siège)
- **Les 5 boules en orbite**: les points du réseau (franchisés, magasins)
- **Les 3 anneaux concentriques**: les ondes de signal qui se propagent
- **Les 5 couleurs**: les 5 piliers de notre identité

### 1.2 Structure du Logo

```
         ● Teal (72°)
        /
       /
  ● Jaune ----[n]---- ● Coral
  (144°)       |       (0°)
               |
       ● Teal  ● Crème
       (216°)  (288°)
```

**Composition:**
- Centre: Cercle avec lettre "n" (Charcoal en light, Crème en dark)
- 3 anneaux concentriques (rayons: 16px, 22px, 28px)
- 5 boules colorées disposées à 72° d'intervalle (360°/5)

### 1.4 Règles d'Usage

#### ✅ À FAIRE

- Utiliser les versions officielles du logo
- Respecter une zone de protection égale à la hauteur du "n"
- Choisir la version adaptée au fond (light/dark)
- Conserver les proportions originales
- Utiliser la version monochrome pour l'impression N&B

#### ❌ À NE PAS FAIRE

- Modifier les couleurs du logo
- Déformer ou étirer le logo
- Ajouter des effets (ombres, contours, dégradés)
- Placer sur un fond trop chargé ou de couleur similaire
- Utiliser une version basse résolution pour l'impression
- Recréer le logo avec d'autres polices

```css
font-family: 'Plus Jakarta Sans', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
```

## 5. Animations du Logo

### 5.1 Animation des Boules (Pulse)

```css
@keyframes logo-pulse {
  0%, 100% {
    transform: scale(1);
    opacity: 1;
  }
  50% {
    transform: scale(1.15);
    opacity: 0.85;
  }
}

.nsignal-logo-animated .dot {
  animation: logo-pulse 2.5s ease-in-out infinite;
}

/* Délais progressifs pour chaque boule */
.dot-1 { animation-delay: 0s; }
.dot-2 { animation-delay: 0.4s; }
.dot-3 { animation-delay: 0.8s; }
.dot-4 { animation-delay: 1.2s; }
.dot-5 { animation-delay: 1.6s; }
```

### 5.2 Animation des Ondes (Propagation)

```css
@keyframes wave-propagate {
  0% {
    stroke-dashoffset: 100;
    opacity: 0.2;
  }
  50% {
    opacity: 0.8;
  }
  100% {
    stroke-dashoffset: 0;
    opacity: 0.4;
  }
}

.nsignal-logo-animated .wave {
  stroke-dasharray: 100;
  animation: wave-propagate 3s ease-in-out infinite;
}
```

### 5.3 Effet Glow au Hover

```css
@keyframes logo-glow {
  0%, 100% {
    filter: drop-shadow(0 0 4px rgba(15, 123, 108, 0.3));
  }
  50% {
    filter: drop-shadow(0 0 12px rgba(15, 123, 108, 0.5));
  }
}

.nsignal-logo:hover {
  animation: logo-glow 1.5s ease-in-out infinite;
}
```
