Usage des Couleurs

Teal - Couleur Principale

Utilisee pour les elements principaux: boutons, liens, accents. Represente la confiance et la stabilite.

Coral - Couleur d'Action

Utilisee pour les CTA secondaires, les notifications, les elements d'attention. Energie et dynamisme.

Jaune Signal - Accent

Utilisee avec parcimonie pour attirer l'attention sur des elements specifiques. Optimisme et innovation.

Charcoal & Creme

Couleurs neutres pour le texte et les fonds. S'inversent entre les modes light et dark.

Regles d'Usage

Zone de Protection

Toujours respecter un espace minimum autour du logo egal a la hauteur du "n".

Bonnes Pratiques

A FAIRE

  • Utiliser les versions officielles du logo
  • Respecter les zones de protection
  • Utiliser le mode adapte au fond (light/dark)
  • Conserver les proportions originales
  • Utiliser la version monochrome si necessaire

A NE PAS FAIRE

  • Modifier les couleurs du logo
  • Deformer ou etirer le logo
  • Ajouter des effets (ombres, contours)
  • Placer sur un fond trop charge
  • Utiliser une version basse resolution
